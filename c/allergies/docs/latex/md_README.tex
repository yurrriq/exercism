Given a person\textquotesingle{}s allergy score, determine whether or not they\textquotesingle{}re allergic to a given item, and their full list of allergies.

An allergy test produces a single numeric score which contains the information about all the allergies the person has (that they were tested for).

The list of items (and their value) that were tested are\+:


\begin{DoxyItemize}
\item eggs (1)
\item peanuts (2)
\item shellfish (4)
\item strawberries (8)
\item tomatoes (16)
\item chocolate (32)
\item pollen (64)
\item cats (128)
\end{DoxyItemize}

So if Tom is allergic to peanuts and chocolate, he gets a score of 34.

Now, given just that score of 34, your program should be able to say\+:


\begin{DoxyItemize}
\item Whether Tom is allergic to any one of those allergens listed above.
\item All the allergens Tom is allergic to.
\end{DoxyItemize}

Note\+: a given score may include allergens {\bfseries{not}} listed above (i.\+e. allergens that score 256, 512, 1024, etc.). Your program should ignore those components of the score. For example, if the allergy score is 257, your program should only report the eggs (1) allergy.\hypertarget{md_README_autotoc_md8}{}\doxysection{\texorpdfstring{Source}{Source}}\label{md_README_autotoc_md8}
