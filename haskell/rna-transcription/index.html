RNA Transcription

Write a program that, given a DNA strand, returns its RNA complement (per RNA transcription).

Both DNA and RNA strands are a sequence of nucleotides.

The four nucleotides found in DNA are adenine (A), cytosine (C), guanine (G) and thymine (T).

The four nucleotides found in RNA are adenine (A), cytosine (C), guanine (G) and uracil (U).

Given a DNA strand, its transcribed RNA strand is formed by replacing each nucleotide with its complement:

  • G -> C
  • C -> G
  • T -> A
  • A -> U

Solution

{-# LANGUAGE LambdaCase #-}

I used the LambdaCase syntactic extension for brevity and to fully adopt the point-free style. 

\case { ... } ==> \x -> case x of { ... }

Apparently it took the Haskell team quite a while to settle on the syntax. 

toRNA :: String -> String
toRNA = map (\case 'G' -> 'C'; 'C' -> 'G'; 'T' -> 'A'; 'A' -> 'U')

I always appreciate a good one-liner.
